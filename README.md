# ECC project

# How to Install from git
```
git clone https://github.com/Stovka/ecc_project
cd ecc_project
python setup.py install

python app.py
```
# How to Install from archive
```
cd ecc_project
python setup.py install

python app.py
```
### Dependencies
App needs pygubu for GUI and pycryptodome for cryptography.
